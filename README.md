# Excel Sales Dashboard

An interactive Excel Sales Dashboard created using Pivot Tables, Charts, and Slicers.

# Project Overview 
This dashboard helps analyze:

- Total Quantity
- Total Cost
- Total Items
- Category-wise Purchases
- Top Buyers
- Supplier Contribution
- Quantity Trends
